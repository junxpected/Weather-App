# Weather App

## Про проект
Це простий та зручний веб-додаток для перевірки погоди в реальному часі. Головна фішка — зовнішній вигляд сайту (фон та відео) змінюється залежно від того, яка погода у вибраному місті.

---

### Що вміє додаток
* **Розумний пошук**: допомагає знайти місто за першими літерами.
* **Живе відео**: показує сонце, дощ або сніг на фоні сторінки.
* **Інтерактивна карта**: автоматично позначає місцеположення міста.
* **Детальна інформація**: показує вологість, вітер та температуру «як відчувається».
* **Прогноз на 5 днів**: короткий огляд погоди на найближчий тиждень.
* **Історія**: запам'ятовує міста, які ви шукали останнім часом.
* **Мобільна версія**: сайт зручно відкривати з телефона.
