# Звіт про проект: Weather App

## Огляд
Цей проект — це веб-додаток для відстеження погоди, який надає актуальні дані про стан атмосфери в будь-якому місті світу. Головна мета проекту — створити не просто інформативний, а візуально привабливий інтерфейс, що реагує на зміни погоди.

## Основний функціонал
* **Пошук та автопідказки**: Користувач може почати вводити назву міста, а система запропонує варіанти для вибору.
* **Динамічний фон**: Фонове відео автоматично змінюється залежно від погоди: сонце, хмари, дощ, сніг або туман.
* **Інтерактивна карта**: Після пошуку міста додаток оновлює карту, показуючи точне місце розташування з маркером.
* **Поточні дані**: Відображення температури, рівня вологості, швидкості вітру та відчутної температури.
* **Прогноз на майбутнє**: Таблиця з прогнозом температури та іконками на наступні 5 днів.
* **Історія запитів**: Список останніх міст, які шукав користувач, для швидкого перемикання між ними.

## Технічні рішення
* **Джерело даних**: Використано OpenWeather API для отримання координат та погодних умов у реальному часі.
* **Картографія**: Інтегровано бібліотеку Leaflet.js для роботи з географічними картами.
* **Збереження даних**: Використано LocalStorage для зберігання історії міст, щоб дані не зникали після закриття вкладки.
* **Дизайн**: Реалізовано стиль Glassmorphism (ефект матового скла), що забезпечує читабельність тексту на будь-якому фоновому відео.

## Висновок
Проект успішно поєднує роботу з API, інтерактивними картами та динамічним контентом, створюючи цілісний та зручний продукт для користувача.
